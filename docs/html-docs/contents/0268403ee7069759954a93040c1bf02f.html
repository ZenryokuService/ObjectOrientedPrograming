ShopScene

UMLClass
TextRpgDiagram :: TextRpgModel :: ShopScene

Description

ストーリーテキストを表示の上、以下の処理を行う。 1. RpgLogic#setItemShopでインスタンスを生成する。 2. <item:XXXXX>(3文字以上、10文字以内)で定義を開始、</item>で定義を終了する。 3. アイテムリストよりアイテム(RpgData)を取得ショップの販売リストに追加する。
